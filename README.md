# GARBO-CCRVibe - Smart Waste Management System

Overview
GARBO is a Smart Waste Management and Monitoring System designed to improve waste collection efficiency through monitoring, reporting, and scheduling.

Features
- User Login System  
- Waste Reporting  
- Collection Schedule Monitoring  
- Incident Reporting  
- Notification System  
- Reports Dashboard  

Purpose
To provide a more efficient and organized waste management system using digital monitoring.

System Type
Web-based application integrated with database for real-time data handling.
